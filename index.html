Esta pagina es para guardar mis codigos de HTML

  • p= --: Para escribir un párrafo.
  • h1,2,3,4,5,6 --: Para cambiar el tamaño de la letra del encabezado.
  • img --: Para poner imágenes
  • src "place holder" --: Insertar una imágen de algún place holder.
  • alt "texto" --: Cuando la imágen no se puede cargar, pone un texto escrito por mi.
  • a href= “a donde quiero ir” --: Para crear hipervínculos
